Create React App#

Create React App

1npx create-react-app my-app

1cd my-app
2npm start # or 'yarn start'

Modules#

  • Webpack: JavaScript bundler
  • Babel: JavaScript complier
  • ES Lint | extension: JavaScript linter
  • Jest: JavaScript test framework
  • Web Vitals: measuring performance, accessibility and best practice

Project Structure#

 1-.github
 2-node_modules
 3-public: has files that Webpack will manage and will also be moved to server when publish
 4-- index.html: the entry point for the browsers. Webpack use this file to inject JavaScript code you write as well as React code and other CSS. <div id="root"></div> is where you application will be loaded.
 5-- index.js: the entry point of the JavaScript application. 'react','react-dom','index.css','App.js'... will be imported in this file.
 6-- app.js
 7-- app.test.js
 8-- app.css
 9-src
10-.gitignore
11-package.json
12-yarn.lock
